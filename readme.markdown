# sxhkd-vim

Fork of kovetskiy/sxhkd-vim on GitHub. Vim stuff for indent, highlight syntax and detect [sxhkd - Simple X hotkey
daemon](https://github.com/baskerville/sxhkd) configuration files.

## Installation

```vim
Plug 'marcandrebrochu/sxhkd-vim'
```
